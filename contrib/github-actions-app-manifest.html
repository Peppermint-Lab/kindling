Register The Kindling GitHub App

This form opens GitHub’s org app registration page for Peppermint-Lab with the Kindling runner permissions already filled in.

  1. Click the button below while signed into GitHub as a Peppermint-Lab admin.
  2. Create the app, then install it on Peppermint-Lab/kindling.
  3. Copy the final browser URL after GitHub redirects you back and paste it into Codex.
The redirect target is https://app.kindling.systems/?kindling_github_app=1. Even if that page looks ordinary, the address bar will include the temporary code=... value I need to finish the setup.
